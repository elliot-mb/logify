## Development guide 

### Prerequisites 

You must have Node Package Manager installed to run this project. 

### Start the frontend

To start the development server:
- Enter root of this repository
- ``cd front``
- ``npm install``
- ``npm run dev -- --open``

The app is now open in developer mode on your browser.
